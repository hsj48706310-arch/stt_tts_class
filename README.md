# 오글거리는 고백 교정기

이 앱은 사용자가 대충 쓴 고백을 입력하면 LLM으로 다듬고, 다양한 목소리로 TTS로 들려줍니다.

## 기능
1. 텍스트 또는 음성으로 고백 입력
2. 감성 가득한 시인 버전, 박력있는 사투리 버전, 깔끔한 직장인 버전으로 교정
3. 다양한 목소리로 TTS 출력

## 설치
1. Python 3.8 이상 설치
2. 의존성 설치: `pip install -r requirements.txt`
3. .env 파일에 OpenAI API 키 추가
4. 실행: `streamlit run feat2.py`

## 사용법
- 앱을 실행하면 입력 방법 선택
- 텍스트 입력 또는 음성 입력
- 스타일 선택 후 교정
- 목소리 선택 후 듣기
