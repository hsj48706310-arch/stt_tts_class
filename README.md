# stt_tts_class

AI Confession Message Refiner

Features
- Speech-to-Text input
- LLM-based message refinement
- Tone/style control
- Text-to-Speech output

- 고백 교정기 : 진심을 전하기 서툰 분들을 위한 재치있는 솔루션을 제공합니다.

1. 사용자가 대충 쓴 진심(예: "나 너 좋아하는데 사귈래?")을 입력합니다.
2. LLM이 이를 '감성 가득한 시인 버전' 혹은 '깔끔한 직장인 버전'으로 다듬어줍니다.
3. 수정된 멘트를 감미로운 목소리(TTS)로 들려줍니다.
